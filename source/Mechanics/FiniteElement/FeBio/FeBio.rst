=====
FEBio
=====
This section contains different tutorials that describe defining and running finite element simulations in FEBio (`febio.org <https://febio.org/>`_).

Most of these tutorials describe how to define the inputs needed to run a simulation without having to use the FEBio GUI (PreView).

The preliminary tutorials demonstrate concepts that are used in more specific applications.
